# Nova Android

Nova Android — приложение для обхода большинства ограничений трафика в России с упором на стабильность, автоматический подбор рабочих стратегий подключения и лёгкости использования.

Приложение ориентировано на реальные нестабильные сети, где условия могут меняться в процессе подключения: Nova умеет адаптироваться к смене сети, подбирать удачные конфигурации, использовать резервные маршруты и сохранять рабочие варианты для следующих запусков.

Видео на Ютубе: https://youtu.be/0oqZ0_x2Dnw

## Возможности

- Автоматический подбор стратегий подключения
- Резервное подключение при недоступности WARP
- Режимы региона: `AUTO`, `WARP`, `EU`, `US`
- Маскировка SNI с автоматическим и ручным выбором домена
- Split tunneling: можно настроить правила для отдельных приложений
- Автоматический реконнект при потере связи или смене сети
- Кэширование и приоритизация успешных конфигураций
- Поддержка автообновления приложения
- Доступ к большинству нейросетей, недоступных из России
- Доступ к большинству сайтов, которые не нравятся Роскомнадзору

## Требования

- Android 9+

В будущем возможно добавление поддержки ТВ-приставок
